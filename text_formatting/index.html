Web Development Terminologies

Web Development Terminologies

HTML
HyperText Markup Language used to structure content on the web.
CSS
Cascading Style Sheets used to design and style HTML elements.
JavaScript
A programming language used to make web pages interactive.
Front-End
The part of a website users interact with directly (HTML, CSS, JS).
Back-End
The server-side of web development that handles databases and logic.
Full Stack
A developer skilled in both front-end and back-end development.
DOM
Document Object Model – a representation of the HTML structure that JavaScript can manipulate.
API
Application Programming Interface – allows communication between software systems.
HTTP
HyperText Transfer Protocol – used for transferring data over the web.
URL
Uniform Resource Locator – the address of a web page.
Responsive Design
Design that adapts to different screen sizes (desktop, tablet, mobile).
Bootstrap
A popular CSS framework for responsive and mobile-first websites.
Semantic HTML
HTML that clearly describes its meaning (e.g., <article>, <section>).
SEO
Search Engine Optimization – improving visibility of websites in search engines.
Git
A version control system for tracking code changes.
GitHub
A web-based platform for hosting and sharing Git repositories.
Node.js
JavaScript runtime built on Chrome's V8 engine for server-side programming.
Framework
A set of tools and libraries that help build web apps faster (e.g., React, Angular).
Database
A structured collection of data (e.g., MySQL, MongoDB).
Hosting
Storing website files on a server to make them accessible online.
